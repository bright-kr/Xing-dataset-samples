# Xing 데이터셋 샘플

<h2>1001개의 Xing 소셜 네트워크 샘플 데이터셋</h2>

![Xing dataset header](https://github.com/bright-kr/Xing-dataset-samples/blob/main/Xing-datasets%20(1).png)

1000개 이상의 레코드로 구성된 Xing 데이터셋 샘플입니다. 데이터셋은 <b>Bright Data API</b>를 사용하여 추출되었습니다.

<h2>Xing 데이터셋에 포함된 일부 데이터 포인트:</h2>

* ```account_id```: 사용자의 계정에 대한 고유 식별자
* ```familyName```: 사용자의 성 또는 성씨
* ```gender```: 사용자의 성별
* ```membership```: 멤버십 정보
* ```country_code```: 국가 코드
* ```experience```: 사용자의 업무 경력에 대한 정보를 포함하는 배열입니다. 배열의 각 항목에는 현재 직무, 회사, 직함, 시작 날짜, 종료 날짜, 회사 페이지, 회사 로고 및 설명과 같은 세부 정보가 포함됩니다.
* ```education```: 사용자의 교육 배경에 대한 정보를 포함합니다.
* ```languages```: 사용자가 알고 있는 언어와 해당 숙련도 수준을 나타냅니다.
* ```groups```: 사용자가 속한 그룹 또는 커뮤니티를 나타냅니다.
* ```interests```: 사용자의 관심사를 지정합니다.

그리고 훨씬 더 많은 정보가 포함됩니다.

이는 "Xing social network"에서 파생된 샘플 하위 집합이며,
<b>9.8M records</b> 이상을 포함하는 데이터셋입니다.

사용 가능한 데이터셋 파일 형식: <b>JSON, NDJSON, JSON Lines, CSV, or Parquet. Optionally, files can be compressed to .gz</b>.

데이터셋 전달 유형 옵션: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>.

업데이트 빈도: <b>Once, Daily, Weekly, Monthly, Quarterly, or Custom basis</b>.

추출된 데이터 포인트에 추가로 제공 가능한 데이터 강화: <b>Based on request.</b>

<b>[Get the full Xing dataset](https://brightdata.co.kr/products/datasets/xing)</b>.

<h2>Xing 데이터셋의 사용 사례는 무엇입니까?</h2>

<h3>1. 리드 생성</h3>
Xing의 최신 직무 및 회사 데이터를 활용하여 리드 생성을 강화하고 CRM 프로세스를 간소화합니다.

<h3>2. 투자 기회</h3>
독일 기업 내부 및 기업 간의 인력 이동을 모니터링하여 새로운 비즈니스 기회를 파악하고 잠재적 투자 대상을 평가합니다.

<h3>3. 우수 인재 채용</h3>
독일어권 국가의 전문 인력에 대한 기술, 교육 및 경력 프로필을 분석하여 우수 인재를 식별하고 추적합니다.
